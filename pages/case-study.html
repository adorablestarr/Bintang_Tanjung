Case Study

Personal Portfolio
Website

Bagaimana saya membangun portfolio profesional dari nol menggunakan HTML, CSS, dan JavaScript murni — tanpa framework, tanpa library eksternal.

Timeline 2 Minggu
Role Full Stack Developer
Status ✅ Live
Tahun 2025
01 — Problem Statement

Masalah yang Ingin Dipecahkan

Sebagai mahasiswa software engineering yang aktif di organisasi dan belajar desain, saya tidak punya satu tempat terpusat untuk menampilkan semua karya dan pencapaian saya kepada recruiter atau kolaborator.

Platform seperti LinkedIn dan GitHub bagus, tapi tidak cukup untuk menampilkan karya desain visual, foto kegiatan organisasi, dan kepribadian profesional saya sekaligus dalam satu tampilan yang kohesif.

"Bagaimana cara membuat portfolio yang bisa menceritakan siapa saya secara profesional dan kreatif, sekaligus mudah di-maintain tanpa backend?"
02 — Solution

Solusi yang Diimplementasikan

Membangun single-page portfolio dengan HTML, CSS, dan JavaScript vanilla yang modular, ringan, dan tidak bergantung framework apapun. Keputusan ini disengaja untuk menunjukkan pemahaman mendalam tentang web fundamentals.

HTML5 Semantic CSS Variables CSS Grid & Flexbox Vanilla JavaScript IntersectionObserver API Clipboard API localStorage CSS Custom Animations

Fitur utama yang diimplementasikan: scroll reveal animation, lightbox image preview, gallery tab switcher, email clipboard copy, dark/light mode toggle dengan localStorage persistence, dan contact form dengan validasi.

05 — Challenges

Tantangan Teknis

🧩
IntersectionObserver & Tab Hidden State Elemen di tab yang hidden (display:none) tidak terdeteksi IntersectionObserver. Solusi: triggerReveal() manual saat tab dibuka, dan DOMContentLoaded trigger untuk tab aktif pertama.
🍞
Toast Notification Stacking Toast menumpuk jika diklik cepat. Solusi: clearTimeout sebelum set timer baru, dan void offsetWidth untuk force CSS reflow agar animasi reset.
🌗
Theme Flash on Load Dark/light mode bisa "flash" sebentar sebelum JS jalan. Solusi: theme.js di-load sebagai script pertama sebelum konten render.
📋
Clipboard API Compatibility navigator.clipboard tidak tersedia di HTTP (hanya HTTPS) dan beberapa browser lama. Solusi: fallback dengan textarea + execCommand.
06 — Lessons Learned

Yang Saya Pelajari

✅ Web fundamentals (HTML/CSS/JS) masih sangat relevan dan powerful tanpa framework
✅ CSS Variables memudahkan theming secara dramatis — dark mode hanya perlu override variabel
✅ IntersectionObserver lebih efisien dari scroll event listener untuk animasi viewport
✅ Modular CSS (satu file per section) membuat debugging jauh lebih mudah
✅ Accessibility (aria-label, role, focus-visible) bukan opsional — ini standar profesional
