🎯 Tujuan Project

Membuat portfolio profesional untuk melamar magang dan pekerjaan sebagai Software Engineer. Ingin ada satu tempat yang menampilkan skill teknis, karya desain, dan kegiatan organisasi secara kohesif.

🛠️ Tech Stack
HTML5 CSS3 Vanilla JavaScript IntersectionObserver CSS Variables Clipboard API localStorage
👩‍💻 Role Saya

Mengerjakan seluruh project secara mandiri — mulai dari desain UI/UX di atas kertas, coding HTML/CSS dari nol, implementasi JavaScript untuk interaktivitas, hingga testing responsiveness di berbagai device.

✨ Fitur Utama

Dark/light mode, scroll reveal animation, lightbox image preview, gallery tab switcher, email copy to clipboard, contact form dengan validasi, dan animated skill progress bars.

⚡ Tantangan & Solusi

Tantangan terbesar adalah membuat animasi scroll reveal bekerja dengan benar di semua skenario — termasuk untuk elemen yang ada di dalam tab yang tersembunyi (display: none).

Bug: IntersectionObserver tidak bisa mendeteksi elemen yang hidden, sehingga kartu gallery di tab kedua tidak pernah mendapat class .visible meskipun sudah di-scroll.

Solusinya: membuat fungsi triggerReveal() yang dijalankan secara manual setiap kali tab dibuka — dengan stagger delay 80ms antar card untuk efek yang natural.

Tantangan lain: toast notification email copy muncul di belakang footer dan bertumpuk kalau diklik cepat. Solusi: pindahkan posisi ke atas layar, tambahkan void offsetWidth untuk force browser reflow sebelum animasi di-reset.

💡 Yang Saya Pelajari

Proyek ini membuktikan bahwa HTML/CSS/JavaScript murni masih sangat powerful tanpa framework. CSS Variables membuat dark mode implementation sangat clean — hanya perlu override variabel di satu tempat.

Saya juga belajar pentingnya accessibility: menambahkan aria-label, focus-visible styling, dan semantic HTML bukan sekedar formalitas, tapi membuat website lebih profesional di mata recruiter teknis.
